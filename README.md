# 🌱 Crop Recommendation System

AI powered crop recommendation system using Machine Learning and Flask.

## Features
- Predicts best crop
- Modern UI/UX
- Dark futuristic theme
- Random Forest ML model
- Flask web application

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML
- CSS

## Input Features
- Nitrogen
- Phosphorous
- Potassium
- Temperature
- Humidity
- Rainfall

## Output
Recommended crop for farming.

## Run Project

```bash
pip install -r requirements.txt
python app.py
```
